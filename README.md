# Coolify Scraper & Volume Backup

This repository provides comprehensive tools for managing Coolify instance data, including automated web scraping capabilities and flexible volume backup solutions designed to ensure your data remains safe and accessible.

## Coolify Scraper

The Coolify Scraper leverages advanced web scraping techniques to extract detailed information from your Coolify instances. It efficiently gathers project details, configurations, and other critical data for backup and migration purposes.

## Volume Backup

The Volume Backup suite offers flexible options for backing up volumes associated with your Coolify instances. Two methods are available to suit different operational needs:

1. **Automated S3 Backups**: Schedule regular, automated backups to AWS S3 buckets using cron jobs, ensuring continuous data protection without manual intervention.
2. **Manual Backup & Restore**: Create on-demand backups and restore them to the same or different Coolify instances, providing flexibility for migrations and disaster recovery scenarios.
